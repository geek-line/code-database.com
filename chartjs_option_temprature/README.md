# chartjs_option_temprature
code-database.comの[Chart.jsで線グラフと棒グラフの美しい雨温図を実装する](https://code-database.com/knowledges/84)で使用したコードを公開しています。

Chart.jsを用いて気温の線グラフと降水量の棒グラフを合わせた複合グラフを作成したコードです。
