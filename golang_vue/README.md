# golang_vue
code-database.comに掲載されている[GolangとVue.jsとChart.jsで小さなSPAアプリを作る](https://code-database.com/knowledges/89)で取り上げたサンプルコードを公開しています。
## コードの概要
Vue.jsとGoで小さなSPAを構築し、雨温図を表示する機能をChart.jsで実装しています。
